3. Testing Commands

// Check active scene count
window.splineControls.getActiveCount()

// Manually pause all scenes
window.splineControls.pauseAll()

// Resume all scenes
window.splineControls.resumeAll()

Important Notes:

  • Update CLOUDFLARE_URL in the script to your deployed URL
  • Canvas IDs must be unique across the page
  • data-spline-scene must match folder names in scenes/
  • Use data-priority="true" for critical scenes that should never pause
  • Desktop: Max 4 active scenes | Mobile: Max 2 active scenes
